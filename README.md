# First custom PCB - Design, Manufacture with JLCPCB, SMD Assembly
Designed and manufactured my first custom PCB using KiCad and JLCPCB. The project involved schematic capture, component selection, PCB layout, design rule checking, 
Gerber generation and manufacturing preperation. This project provided hands-on experience with the end-to-end PCB development workflow.
## Board Design in KiCad
The PCB itself is only a simple USB-C powered 4-digit display controlled by an STM32g030. This was kept simple and only done in an afternoon as this was more for the
experience of getting it made and assembled, rather than a fully complete project. Therefore the design itself is fairly rudimentary.\
The board consists of two layers, with component placement all on the top layer.
